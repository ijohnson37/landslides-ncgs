Accessibility Statement

Our commitment to making the Debris Flow Early Notification System usable for everyone.

Standard we target

The DEFNS dashboard targets WCAG 2.1 Level AA conformance, the standard most commonly referenced for compliance with the Americans with Disabilities Act (ADA) and Section 508 of the U.S. Rehabilitation Act.

What we’ve done

Perceivable

  • All text has a contrast ratio of at least 4.5:1 against its background (large text 3:1).
  • Precipitation categories are encoded with both color and text labels, so the dashboard remains usable for users with any form of color vision deficiency.
  • The interactive map is supplemented by an accessible HTML table (“Flagged debris flow zones”), which carries the same information in a screen-reader-readable form.
  • All images have meaningful alternative text.

Operable

  • All interactive controls (sliders, radio buttons, checkboxes, buttons) are reachable and operable with a keyboard alone.
  • A “Skip to alerts table” link is available at the top of every page for keyboard users.
  • Focus is always visible: a high-contrast gold outline indicates the currently focused element.
  • Auto-refresh can be paused via the “Auto-refresh” checkbox in the sidebar.
  • Interactive targets meet the 44×44 pixel minimum size for comfortable use on touch devices.

Understandable

  • Each control has a clear label, and additional help text explains technical terms like “NDFD category.”
  • The page language is declared as English; consistent navigation and headings are used across pages.

Robust

  • The dashboard uses valid, semantic HTML5 (proper heading hierarchy, table headers, form labels) so assistive technologies can interpret it reliably.
  • Dynamic updates (e.g., alert counts changing after a refresh) use ARIA live regions to be announced by screen readers without interrupting the user.

Known limitations

  • The Leaflet map itself is a visual aid. Although the map container has descriptive labels, screen-reader users should rely on the alerts table for canonical data about which zones are flagged.
  • The NC Geological Survey debris flow reference layer is loaded from an external ArcGIS Online service; if that service is slow, some data may take time to appear. The alert detection itself does not depend on this layer rendering.

Report an accessibility issue

If you encounter an accessibility barrier on this site, please contact the NC Geological Survey landslides team (landslidesncgs.com) and describe the problem so we can address it. We aim to respond to accessibility reports within 10 business days.

Last reviewed

This statement reflects the dashboard’s status as of its initial launch. It will be updated when significant changes are made to the site.
